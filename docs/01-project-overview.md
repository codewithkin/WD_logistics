# WD Logistics - Project Overview

## Company Profile
WD Logistics is a freight transportation company that manages trucks, drivers, and cargo deliveries for customers.

## Project Goal
Build an MVP/prototype system to showcase the core functionality needed for managing:
- Freight and cargo
- Orders and deliveries
- Staff and drivers
- Basic accounting

---

## Technology Stack

| Component | Technology |
|-----------|------------|
| Web Frontend | Next.js |
| Backend API | Hono |
| Mobile Apps | Expo + React Native |

---

## Platform Distribution

### Mobile Apps (Expo/React Native)
- **Admin App** - Full system oversight and management
- **Driver App** - Delivery management and updates

### Web Application (Next.js)
- **Customer Portal** - Order placement and tracking
- **Staff Dashboard** - Operations management
- **Accountant Dashboard** - Financial overview

---

## User Roles

| Role | Platform | Primary Responsibility |
|------|----------|----------------------|
| Admin | Mobile | Full system control, oversight of all operations |
| Driver | Mobile | Delivery execution, status updates |
| Staff | Web | Order processing, scheduling, customer support |
| Accountant | Web | Financial records, invoicing, payments |
| Customer | Web | Place orders, track shipments |

---

## MVP Scope

This prototype will include only essential features needed for:
1. Basic user authentication and role-based access
2. Order creation and management
3. Freight/cargo tracking
4. Driver assignment and delivery updates
5. Simple invoicing and payment tracking

**Out of Scope for MVP:**
- Advanced reporting and analytics
- Real-time GPS tracking
- Automated route optimization
- Integration with external systems
- Notifications (push/email/SMS)
- Document management
- Customer reviews/ratings
