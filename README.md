# WD Logistics System

A logistics management system for freight transportation with role-based access for Admin, Staff, Drivers, Accountants, and Customers.

## Tech Stack

- **Backend**: Hono
- **Web App**: Next.js
- **Mobile Apps**: Expo + React Native

## Applications

| Application | Platform | Users |
|-------------|----------|-------|
| Admin App | Mobile (iOS/Android) | Administrators |
| Driver App | Mobile (iOS/Android) | Drivers |
| Web Portal | Browser | Staff, Accountants, Customers |
